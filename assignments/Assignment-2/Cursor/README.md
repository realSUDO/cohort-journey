## Sections Recreated

- **Navigation Bar** - Sticky header with logo animation on hover, navigation links, and CTA buttons
- **Hero Section** - Large heading with gradient text, download button, and product screenshot
- **Company Section** - Trusted company logos grid
- **Feature Sections** - Three alternating layout feature cards showcasing Agent, Tab autocomplete, and ecosystem
- **Testimonials Section** - 6-card grid with quotes from Diana Hu (YC), shadcn, Andrej Karpathy, Patrick Collison (Stripe), ThePrimeagen, and Greg Brockman (OpenAI)
- **Frontier Section** - Three-card grid highlighting model access, codebase understanding, and enterprise features
- **Changelog Section** - Recent updates with version badges and dates
- **Secondary CTA** - Team photo section with join CTA
- **Recent Highlights** - Sidebar layout with blog post previews
- **Download CTA** - Large centered download section
- **Footer** - Multi-column footer with product, resources, company, legal, and connect links, plus theme toggle and language picker
