# Mintu - Mintlify Website Clone

A close to pixel-perfect recreation of Mintlify's landing page, built with vanilla HTML and CSS.

**Live Site:** [mintu-five.vercel.app](https://mintu-five.vercel.app/)

## Sections Recreated

- **Navigation Bar** - Fixed header with blur effect and responsive buttons
- **Hero Section** - Animated badge, gradient background, email signup form, and hero image
- **Sponsors Section** - Logo grid featuring Anthropic, Coinbase, Microsoft, Perplexity, HubSpot, X, PayPal, and Lovable
- **Features Section** - Three feature cards with embedded videos showcasing LLMs/TXT/MCP, Agent, and Assistant capabilities
- **Enterprise Section** - Multi-part layout with partnership info, compliance details, customer story, and brand logos
- **Customer Stories** - Interactive card carousel with Perplexity, X, and Kalshi case studies
- **CTA Section** - Call-to-action with pricing and quickstart cards
- **Footer** - Complete footer with navigation links, social icons, security badges, and theme toggle
